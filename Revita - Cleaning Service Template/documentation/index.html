Structure

 All the directories and files are well organized as it shown on the image bellow.

HTML File Structure

This template is a mobile friendly (responsive) layout which is designed based on Twitter Bootstrap v4 . See our HTML file structure which is in index.html. Your contents goes inside container.

CSS File Structure

We are using 16 CSS files across template and comments in capital letter describe the linked stylesheet bellow. We separated styles for improving convenience to edit this template. If you're going to use this template in production I'd recommend you to merge all stylesheets in one file (for decreasing http queries). 

<!-- CSS Begins

Animations



	

		We used WOW plugin to trigger animations on scroll. It used Animate.css for CSS animation. You will find all of the animations in animation.css website. To learn more about WOW please read their animation.css.

	

		Example:

	
<div class="row wow fadeInUp" data-wow-delay="0.5s">
         Your Content Goes Here
</div><!-- /. row -->
